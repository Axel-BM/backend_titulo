## Para poder levantar el servicio, primero que todo debe ejecutar el comando 

```sh
npm install
```

## Una vez que se han hecho las instalaciones con el comando 
```sh
npm run dev
```
## levantará el servicio correspondiente al Backend.

## IMPORTANTE!! DEBE TENER ACTIVO EL SERVIDOR DE BASES DE DATOS, SE RECOMIENDA LA INSTALACIÓN DE XAMPP PARA SUBIR EL SCRIPT SQL
## LA BASE DE DATOS DEBE SER CREADA CON EL COTEJAMIENTO
```sh
utf8mb4_0900_ai_ci
```
